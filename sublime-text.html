Sublime Text

Na minha opinião, é o melhor editor de HTML e CSS.

Baixe gratuitamente no site: www.sublimetext.com

Se você viu os códigos HTML, CSS e Materialize, faça uma cópia dos arquivos deste site e abra no Sublime Text.

Experimente alterar os textos, cores e imagens destas páginas.

Incluí comentários nos códigos HTML, para facilitar o seu entendimento.

Quando conhece os comandos HTML, CSS e Materialize, você pode criar qualquer tipo página !

Use essa versão do Materialize (versão 1.0.0 ) que você já baixou e começar a criar as suas páginas!
